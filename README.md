# Customer Behavior Analysis Dashboard

## 📊 Dashboard Preview!

[Dashboard](dashboard.jpg)


## 📊 Project Overview

This project analyzes customer purchasing behavior using real-world e-commerce data. The goal is to identify patterns in customer activity, revenue distribution, and regional performance.

## 🛠️ Tools Used

* Python (Pandas)
* Power BI

## 🔍 Key Insights

* Major cities like São Paulo generate the highest revenue due to a higher number of orders.
* Revenue is strongly correlated with order volume rather than higher spending per customer.
* Smaller cities show higher average spending per customer, indicating potential high-value segments.

## 📈 Dashboard Features

* Total Revenue KPI
* Revenue by City
* Number of Orders by City

## 📁 Files Included

* `final_data_sample.csv`: Sample of cleaned dataset
* `Customer Behavior Analysis.pbix`: Power BI dashboard
* `Customer Behavior Analysis.ipynb`: Data analysis notebook

## 🚀 Conclusion

This analysis highlights how customer distribution impacts revenue and provides insights that can support marketing and sales strategies.
